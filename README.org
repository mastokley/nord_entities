Basic usage:

Create and enter a virtual python 3 environment:
src_{virtualenv -p $(which python3) ~/.virtualenvs/nord_entities}
src_{. ~/.virtualenvs/nord_entities/bin/activate}

Install requirements:
src_{pip install -r requirements.txt}

Run script against an api endpoint:
src_{python -m extract_entities /v1/search?query=cats}

The script will hit the endpoint and write a file named "entities/[narrative_id]" for each narrative.
